# Clipboard Cleaner
An android app for showing and cleaning clipboard.

## 🛠️ Technologies Used 
- Kotlin
- Jetpack Compose

## ⚠️ Note
On some android devices the clipboard cleaning might not work as some vendors implement their own method of managing clipboards.
